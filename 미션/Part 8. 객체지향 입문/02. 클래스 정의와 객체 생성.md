### **미션 제목**: 나만의 `Book` 클래스 정의하기

### **미션 내용**
책 한 권의 정보를 담을 수 있는 `Book` 클래스를 정의합니다. 이 클래스는 책의 `제목`, `저자`, `페이지 수`를 속성으로 가집니다. 클래스 정의가 완료되면, 이 `Book` 클래스를 사용하여 두 권의 서로 다른 책 객체(인스턴스)를 생성하고, 각 객체에 저장된 정보를 형식에 맞게 출력합니다.

### **미션 요구 사항**
1.  클래스 이름은 `Book`으로 정의합니다.
2.  객체를 생성할 때 `제목`, `저자`, `페이지 수`를 인자로 전달받아 초기화해야 합니다.
3.  `__init__` 메서드를 사용하여 객체의 속성(`title`, `author`, `pages`)을 설정합니다.
4.  아래 두 정보를 가지는 `Book` 인스턴스를 각각 생성합니다.
    - 책 1: `파이썬 정복` (저자: 김파이, 페이지: 300)
    - 책 2: `알고리즘 마스터` (저자: 이알고, 페이지: 450)
5.  생성된 각 인스턴스의 정보를 f-string을 사용하여 `"제목: [제목], 저자: [저자], 페이지: [페이지]쪽"` 형식으로 출력합니다.
6.  (선택) 첫 번째로 생성한 책 객체의 페이지 수를 320으로 변경한 후, 두 책 객체의 정보를 다시 출력하여 객체 간 데이터가 독립적으로 관리되는지 확인합니다.

### **미션 풀이**
이 미션은 객체지향 프로그래밍의 기본 개념인 클래스 정의와 인스턴스 생성을 연습하는 것을 목표로 합니다. 미션 해결 과정은 다음과 같은 단계로 구성됩니다.

1.  **`Book` 클래스 정의**:
    - `class Book:` 구문을 사용하여 클래스를 선언합니다.
    - 객체가 생성될 때 `제목`, `저자`, `페이지 수`를 필수로 받기 위해 `__init__` 특별 메서드를 정의합니다.
    - `__init__` 메서드 내에서 `self.title = title`과 같이 매개변수로 받은 값들을 인스턴스의 속성으로 할당하여 각 객체가 고유한 데이터를 저장하도록 합니다.

2.  **`Book` 인스턴스 생성**:
    - 정의된 `Book` 클래스를 바탕으로 `book1`과 `book2` 변수에 두 개의 서로 다른 인스턴스를 생성합니다.
    - 생성 시 `__init__` 메서드에 정의된 `title`, `author`, `pages` 인자를 순서에 맞게 전달합니다.

3.  **정보 출력**:
    - `print()` 함수와 f-string을 사용하여 각 인스턴스의 정보를 출력합니다.
    - 객체의 속성에 접근하기 위해 `book1.title`, `book2.author`와 같이 점(`.`) 연산자를 사용합니다.

4.  **(선택) 속성 변경 및 독립성 확인**:
    - `book1.pages = 320` 코드를 통해 첫 번째 책 객체의 페이지 수 속성 값을 직접 변경합니다.
    - 두 책 객체의 정보를 다시 출력하여 `book1`의 변경 사항이 `book2`에 영향을 미치지 않음을 확인합니다. 이는 각 인스턴스가 독립적인 데이터를 가진다는 것을 증명합니다.

#### **최종 전체 소스코드**
```python
# Book 클래스 정의
class Book:
    def __init__(self, title, author, pages):
        self.title = title
        self.author = author
        self.pages = pages

# 두 개의 Book 인스턴스 생성
book1 = Book("파이썬 정복", "김파이", 300)
book2 = Book("알고리즘 마스터", "이알고", 450)

# 생성된 인스턴스 정보 출력
print("--- 초기 정보 ---")
print(f"제목: {book1.title}, 저자: {book1.author}, 페이지: {book1.pages}쪽")
print(f"제목: {book2.title}, 저자: {book2.author}, 페이지: {book2.pages}쪽")

# (선택) 첫 번째 책의 페이지 수 변경 및 독립성 확인
print("\n--- 페이지 변경 후 정보 ---")
book1.pages = 320
print(f"제목: {book1.title}, 저자: {book1.author}, 페이지: {book1.pages}쪽")
print(f"제목: {book2.title}, 저자: {book2.author}, 페이지: {book2.pages}쪽")
```
